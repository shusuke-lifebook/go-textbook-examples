# 第2章 変数・型・演算子

「Go言語の教科書」第2章のサンプルコードです。

## 実行方法

```bash
go run main.go
```

## 内容

- `var` と `:=` による変数宣言
- ゼロ値の確認
- 型変換（`int` → `float64` → `uint`）
- `const` と `iota` による定数宣言
- 基本的な演算子
